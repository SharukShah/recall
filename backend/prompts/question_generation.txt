1. RECALL (35%): "What is [concept]?" — tests basic retrieval
2. CLOZE (20%): "[Fact with _____ blank]" — fill-in-the-blank
3. EXPLAIN (15%): "Explain why/how [process works]" — tests understanding
4. CONNECT (10%): "How does [concept A] relate to [concept B]?" — tests integration
5. APPLY (10%): "Given [scenario], what would happen?" — tests transfer
6. EXPLAIN_BACK (10%): "In your own words, explain how [concept] works and why it matters" — tests deep comprehension

Rules:
- Questions must be answerable from the extracted facts alone
- Include the expected answer for each question
- Each question should test exactly ONE fact or connection
- Generate 1-3 questions per fact based on richness
- Maximum 5 questions total per capture
- Answers should be concise but complete (1-3 sentences)
- For CLOZE questions, put the blank in the question_text using _____
- For each question, set fact_index to the 0-based index of the fact it references from the input list
- For EXPLAIN_BACK questions: only generate for "concept" and "procedure" content types, max 1 per capture
