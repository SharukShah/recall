# Extension Icon Placeholders

Since we cannot generate actual PNG images in this environment, you'll need to create icons manually or use an icon generator.

## Icon Requirements

Create three PNG icon files with the following dimensions:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Suggested Icon Design

**Theme:** Brain/Memory symbol with blue color scheme (matching ReCall brand)

**Options:**
1. **Simple approach:** Use a brain emoji or Unicode symbol
   - Download from https://www.iconfinder.com/ or https://www.flaticon.com/
   - Search for "brain icon" or "memory icon"
   - Choose blue/purple color scheme
   - Export as PNG in required sizes

2. **Quick generator:**
   - Visit https://www.favicon-generator.org/
   - Upload a simple image or logo
   - Generate all required sizes
   - Save to `extension/icons/` directory

3. **Temporary placeholder (text-based):**
   - Use any simple blue square PNG
   - Name appropriately (icon16.png, icon48.png, icon128.png)
   - Replace later with proper icons

## To Create Manually

If you have image editing software (Photoshop, GIMP, etc.):
1. Create a 128x128 canvas with blue background (#3b82f6)
2. Add white text "R" (for ReCall) centered
3. Export as PNG
4. Resize to 48x48 and 16x16 for other sizes

The extension will work with any PNG files of the correct dimensions.
